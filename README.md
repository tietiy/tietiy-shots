# tietiy-shots

Transient screenshots for review. Pushed automatically by the operator's local
`watch-shots.sh` watcher.

**⚠️ PUBLIC REPO — git history is forever.**
Only chart / code / build screenshots go here. Never portfolio data, account
values, P&L, API keys, or anything else operator-sensitive.

Files in `shots/` are accessed via raw URLs:
`https://raw.githubusercontent.com/tietiy/tietiy-shots/main/shots/<filename>`
